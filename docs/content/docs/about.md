ustam は、Empirical Software Engineering(実証的ソフトウェア工学)の講義課題として開発されている
Rust製CLIツールです。単なる `ls` の実装にとどまらず、ドキュメンテーション・テスト・CI/CD・Docker・
リリース自動化といったソフトウェアエンジニアリングの実践を目的としています。
