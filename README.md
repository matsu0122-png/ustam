# ustam

## Tagline（1行概要）
Rustで実装した拡張版 `ls` コマンド風CLIツール

---

## 概要（Overview）
本プロジェクトは、Unixの `ls` コマンドの基本機能を拡張したCLIツールです。  
指定したディレクトリ、またはデフォルトで現在のディレクトリ内のファイルやディレクトリ一覧を表示します。

本ツールは、CLIツールとしての基本設計およびUNIX哲学に基づき、シンプルかつ拡張性の高い構成を目指しています。

また、以下のような拡張機能の実装を目的としています。

- ファイルサイズの人間可読形式表示（例：1.2KB, 1.2MB）
- ディレクトリ内にREADME.mdが存在する場合、Taglineの表示
- PDFファイルのタイトル表示
- `.gitignore` などを考慮した表示制御
- 最近作成されたファイルの強調表示
- ソート機能（ファイルサイズ・更新日時・名前）

本プロジェクトは、ソフトウェア工学の講義課題として、  
CLI設計・Rustによる実装・README作成の理解を目的として開発しています。
